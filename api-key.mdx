title: Getting Your GPT-trainer API Key
---
In this comprehensive resource, 
you'll find step-by-step instructions on how to request and receive your API key, enabling you 
to unlock the full potential of the GPT-trainer API for your projects. Whether you're a developer, 
researcher, or enthusiast, this page will walk you through the process, ensuring you have all 
the necessary information to get started. If you already have your API key, you can also explore 
practical examples and integration tips to kickstart your journey with GPT-trainer. Let's begin!

## Step 1: Access GPT-trainer Platform

1. Go to the [GPT-trainer platform](https://app.gpt-trainer.com).

## Step 2: Create an Account (If Not Already Created)

2. If you don't have an account on the GPT-trainer platform, click on the "Sign Up" or "Create Account" 
button and follow the registration process. Provide the required information to create your account.

## Step 3: Access Your Account Menu

3. After successfully logging in or if you already have an account, you'll be directed to the platform. 
Look for the "Profile" menu in the platform's navigation bar. Click on it to access your account settings.

<Frame>
  <img src="/images/api-key-1.png" />
</Frame>

## Step 4: Navigate to API Keys Section

4. Inside your account settings, you will find an "API Keys" section. Within the API Keys section, you should see a
n option to "Generate new key" button. Click on it to initiate the process of creating a new API key.

<Frame>
  <img src="/images/api-key-2.png" />
</Frame>

## Step 5: Enter Key Description and Generate

In the Create Key dialog, provide a description for your API key to help identify its purpose. 
Once you've entered the description, click the "Generate" button to create your API key.

<Warning>
    Be sure to copy and save your API key in a secure location. Do not share it with anyone, as it 
    provides access to the GPT-trainer API services. Treat your API key like a password.
</Warning>

Now, you've successfully requested, generated, and saved your GPT-trainer API key. 
You can use this key to authenticate your requests and integrate GPT-trainer into your applications, scripts, or projects.
