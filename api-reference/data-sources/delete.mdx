<RequestExample>

```bash Curl
curl --location --request POST 'https://app.gpt-trainer.com/api/v1/data-source/{uuid}/delete' \
--header 'Authorization: Bearer <token>'
```

```py Python
import requests

url = 'https://app.gpt-trainer.com/api/v1/data-source/{uuid}/delete'
headers = {
    'Authorization': 'Bearer <token>'
}

params = {
    'uuid': '<source-uuid>'
}

response = requests.POST(url, headers=headers, params=params)

if response.status_code == 200:
    print("Request successful! Source deleted.")
else:
    print("Request failed with status code:", response.status_code)
    print(response.text)
```

```ts JavaScript
const axios = require('axios');

const url = 'https://app.gpt-trainer.com/api/v1/data-source/{uuid}/delete';
const headers = {
    'Authorization': 'Bearer <token>'
};

const params = {
    uuid: '<source-uuid>'
};

axios.post(url, { headers, params })
    .then(response => {
        console.log('Request successful! SOurce deleted.');
    })
    .catch(error => {
        console.error('Request failed:', error);
    });
```
