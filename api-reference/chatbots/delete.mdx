<ParamField query="uuid" type="string" required/>

### Response

<ResponseField name="success" type="boolean">
  Indicates if api call was successful
</ResponseField>


<RequestExample>

```bash Curl
curl --location --request DELETE 'https://app.gpt-trainer.com/api/v1/chatbot/{uuid}/delete' \
--header 'Authorization: Bearer <token>'
```

```bash Python
import requests

url = 'https://app.gpt-trainer.com/api/v1/chatbot/{uuid}/delete'
headers = {
    'Authorization': 'Bearer <token>'
}

params = {
    'uuid': '<chatbot-uuid>'
}

response = requests.delete(url, headers=headers, params=params)

if response.status_code == 200:
    print("Request successful! Chatbot deleted.")
else:
    print("Request failed with status code:", response.status_code)
    print(response.text)
```

```bash JavaScript
const axios = require('axios');

const url = 'https://app.gpt-trainer.com/api/v1/chatbot/{uuid}/delete';
const headers = {
    'Authorization': 'Bearer <token>'
};

const params = {
    uuid: '<chatbot-uuid>'
};

axios.delete(url, { headers, params })
    .then(response => {
        console.log('Request successful! Chatbot deleted.');
    })
    .catch(error => {
        console.error('Request failed:', error);
    });
```
